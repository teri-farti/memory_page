# Memory

This project is a small, atmospheric symbol-matching game, created in a single day using the Godot 3 engine. The focus is on simplicity, a relaxed pace, and a cozy visual atmosphere, rather than mechanical complexity.

---

# Overview

The game features a quiet, minimalist board with flipped cards. The player flips pairs, trying to find matching symbols, while the environment subtly shifts tone. Smooth transitions, soft sound effects, and a warm color palette create a meditative feel, transforming classic mechanics into a relaxing micro-journey.

---

# Key Features

## Classic Memory gameplay:

- Flip cards, find pairs, and clear the board.

## Atmospheric design:

- Soft lighting, smooth animation, and a calm color scheme create a tranquil atmosphere.

## Responsive Interaction:

- Subtle UI feedback, card flip animations, and subtle sound effects enhance the tactile sensation of every action.

## Lightweight Structure:

- A compact project, created in a single day, with an emphasis on quality and atmosphere over scale.

---

# Project Status

## Completed:

- Core gameplay loop, card logic, animations, sound effects, difficulty modes, and basic user interface.

## Future Plans:

- Additional card themes and background effects were excluded due to the development time limit (one day).

# Technical Specifications

- Engine: Godot 3

- Language: GDScript

- Rendering: 2D

- Input: Mouse and touchscreen support
